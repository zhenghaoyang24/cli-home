## Vue3 CLI浏览器主页应用 - 开发文档

---

### 1. 项目概述

本项目是一个基于 Vue3 全家桶的 CLI 形式浏览器主页应用，旨在提供一个视觉吸引力强的终端模拟器界面，集成搜索、AI对话和快捷指令功能，为用户提供高效的浏览器入口体验。

---

### 2. 技术栈选型

| 分类 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 框架 | Vue | 3.4.x | 核心框架，使用 Composition API |
| 语言 | TypeScript | 5.3.x | 类型安全的开发语言 |

### 4. 核心功能模块设计

#### 4.1 CLI交互界面

**功能描述**：实现终端模拟器，支持彩色文本输出和命令高亮。

**实现思路**：

1. **终端显示**：
   - 使用 `<pre>` 标签展示终端内容
   - 支持 ANSI 转义码解析，实现彩色文本
   - 命令输入区域固定在底部

2. **命令解析**：
   - 解析用户输入的命令
   - 支持命令高亮（不同命令类型显示不同颜色）
   - 实现命令历史记录

3. **输出格式化**：
   - 支持多种输出类型（普通文本、成功、错误、警告）
   - 实现打字机效果的文本输出

**核心代码结构**：

```typescript
// 命令类型定义
interface Command {
  type: 'search' | 'ai' | 'shortcut' | 'config' | 'help';
  args: string[];
  raw: string;
}

// 输出行类型
interface OutputLine {
  id: string;
  type: 'input' | 'output' | 'success' | 'error' | 'warning';
  content: string;
  timestamp: Date;
}
```

#### 4.2 搜索功能

**功能描述**：通过CLI输入搜索关键词并执行搜索。

**实现思路**：

1. **搜索命令格式**：
   ```
   search [关键词]        // 使用默认搜索引擎搜索
   search -e <引擎> [关键词]  // 指定搜索引擎
   config search set <key> <value>  // 配置搜索参数
   ```

2. **搜索引擎配置**：
   - 支持多种搜索引擎（Google、Bing、百度等）
   - 可配置默认搜索引擎
   - 支持自定义搜索引擎

3. **搜索执行**：
   - 构建搜索URL
   - 在新标签页打开搜索结果

**搜索服务接口**：

```typescript
interface SearchEngine {
  id: string;
  name: string;
  url: string;      // 搜索URL模板，如 https://www.google.com/search?q={query}
  icon: string;
}

interface SearchConfig {
  defaultEngine: string;
  engines: SearchEngine[];
}
```

#### 4.3 AI对话功能

**功能描述**：终端内发起AI对话，支持配置API参数。

**实现思路**：

1. **AI命令格式**：
   ```
   ai [问题]             // 发起AI对话
   ai config set <key> <value>  // 配置AI参数
   ai clear             // 清空对话历史
   ```

2. **AI配置参数**：
   - API密钥
   - 请求URL
   - 模型名称
   - 温度参数
   - 最大token数

3. **对话流程**：
   - 发送请求到AI服务
   - 流式接收响应
   - 显示打字机效果

**AI服务接口**：

```typescript
interface AIConfig {
  apiKey: string;
  apiUrl: string;
  model: string;
  temperature: number;
  maxTokens: number;
}

interface AIMessage {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
}
```

#### 4.4 快捷指令系统

**功能描述**：可配置的快捷指令，输入指令直接导航。

**实现思路**：

1. **快捷指令命令格式**：
   ```
   [指令名]             // 直接执行指令（如 bilibili）
   shortcut list        // 列出所有指令
   shortcut add <name> <url>  // 添加指令
   shortcut edit <name> <url> // 修改指令
   shortcut delete <name>    // 删除指令
   ```

2. **指令管理**：
   - 支持添加、修改、删除
   - 支持模糊匹配
   - 本地持久化存储

**快捷指令接口**：

```typescript
interface Shortcut {
  id: string;
  name: string;
  url: string;
  icon?: string;
  createdAt: Date;
}
```

### 5. API接口设计

#### 5.1 搜索服务

| 方法 | 功能 | 参数 | 返回值 |
|------|------|------|--------|
| `buildSearchUrl` | 构建搜索URL | `engineId: string`, `query: string` | `string` |
| `getEngines` | 获取所有搜索引擎 | 无 | `SearchEngine[]` |
| `setDefaultEngine` | 设置默认引擎 | `engineId: string` | `void` |
| `addEngine` | 添加搜索引擎 | `engine: Omit<SearchEngine, 'id'>` | `SearchEngine` |
| `removeEngine` | 删除搜索引擎 | `engineId: string` | `void` |

#### 5.2 AI服务

| 方法 | 功能 | 参数 | 返回值 |
|------|------|------|--------|
| `sendMessage` | 发送消息到AI | `message: string` | `Promise<string>` |
| `sendMessageStream` | 流式发送消息 | `message: string`, `callback: (chunk: string) => void` | `Promise<void>` |
| `clearHistory` | 清空对话历史 | 无 | `void` |
| `getHistory` | 获取对话历史 | 无 | `AIMessage[]` |

#### 5.3 存储服务

| 方法 | 功能 | 参数 | 返回值 |
|------|------|------|--------|
| `saveConfig` | 保存配置 | `key: string`, `value: any` | `void` |
| `loadConfig` | 加载配置 | `key: string`, `defaultValue: any` | `any` |
| `saveShortcuts` | 保存快捷指令 | `shortcuts: Shortcut[]` | `void` |
| `loadShortcuts` | 加载快捷指令 | 无 | `Shortcut[]` |

### 6. 状态管理方案

#### 6.1 Pinia Stores

**1. Terminal Store** (`stores/terminal.ts`)

```typescript
interface TerminalState {
  history: OutputLine[];
  currentInput: string;
  commandHistory: string[];
  historyIndex: number;
}

// Actions
- addOutput(content: string, type: OutputLine['type'])
- executeCommand(command: string)
- clearHistory()
- navigateHistory(direction: 'up' | 'down')
```

**2. Search Store** (`stores/search.ts`)

```typescript
interface SearchState {
  config: SearchConfig;
}

// Actions
- setDefaultEngine(engineId: string)
- addEngine(engine: Omit<SearchEngine, 'id'>)
- removeEngine(engineId: string)
- executeSearch(query: string, engineId?: string)
```

**3. AI Store** (`stores/ai.ts`)

```typescript
interface AIState {
  config: AIConfig;
  messages: AIMessage[];
  isLoading: boolean;
}

// Actions
- updateConfig(key: keyof AIConfig, value: any)
- sendMessage(message: string)
- clearMessages()
```

**4. Shortcuts Store** (`stores/shortcuts.ts`)

```typescript
interface ShortcutsState {
  shortcuts: Shortcut[];
}

// Actions
- addShortcut(name: string, url: string)
- updateShortcut(id: string, name: string, url: string)
- deleteShortcut(id: string)
- executeShortcut(name: string)
- findShortcut(name: string)
```

### 7. UI/UX设计规范

#### 7.1 设计原则

- **深色主题**：黑色背景（`#0d1117`），终端风格
- **彩色文本**：支持多种颜色输出（绿色成功、红色错误、黄色警告、蓝色信息）
- **等宽字体**：使用 `JetBrains Mono` 或 `Fira Code` 等编程字体

#### 7.2 颜色规范

| 用途 | 颜色 | 说明 |
|------|------|------|
| 背景 | `#0d1117` | 终端背景色 |
| 输入文本 | `#e6edf3` | 用户输入颜色 |
| 输出文本 | `#8b949e` | 普通输出颜色 |
| 成功 | `#3fb950` | 成功消息 |
| 错误 | `#f85149` | 错误消息 |
| 警告 | `#d29922` | 警告消息 |
| 命令高亮 | `#58a6ff` | 命令关键字 |
| 光标 | `#e6edf3` | 输入光标 |

#### 7.3 交互规范

1. **命令输入**：
   - Enter 键执行命令
   - Tab 键自动补全
   - 上下箭头翻阅历史

2. **输出动画**：
   - 打字机效果输出文本
   - 命令执行时有加载动画

3. **响应式设计**：
   - 支持移动端和桌面端
   - 终端宽度自适应

### 14. 安全性考虑

1. **API密钥保护**：
   - API密钥仅存储在本地，不上传服务器
   - 不记录到命令历史中
   - 输入时进行掩码显示

2. **XSS防护**：
   - 对用户输入进行HTML转义
   - 对AI返回内容进行安全过滤

3. **链接安全**：
   - 验证快捷指令URL格式
   - 禁止执行javascript协议链接
